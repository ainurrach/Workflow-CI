# Bukti Monitoring Prometheus

Folder ini berisi bukti bahwa sistem telah dimonitor menggunakan Prometheus.

## Bukti yang disertakan
- Screenshot Prometheus Targets (UP)
- Screenshot grafik metrik (CPU / Request / Custom metric)
- Exporter berjalan dengan baik

Monitoring dilakukan untuk memastikan layanan ML berjalan secara observability-ready.
